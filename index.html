Benvingut/da a l'entorn d'entrenament per la iniciació a la programació

Aquí trobaràs diferents tips i recursos per començar el teu aprenentatge en programació amb l'ajuda d'OpenAI i l'assessoria personalitzada d'OfficeGirbau i l'acadèmia Woow.

Programes necessaris

Per poder entrenar hauràs de tenir instal·lats els següents programes:

  • Visual Studio Code
  • Python

Accés a OpenAI

Per accedir a OpenAI i aprofitar el seu entorn d'entrenament, segueix els següents passos:

  1. Visita el lloc web d' OpenAI: https://www.openai.com/
  2. Crea un compte en OpenAI y registra't si encara no en tens una (no té cap cost al no ser que vulguis la versió PRO).
  3. Inicia sessió en el teu compte d'OpenAI. El registre també es pot fer amb compte de Google o de Microsoft.
  4. Explora els recursos, documentació i eines disponibles per l'aprenentatge i entrenament en programació.
